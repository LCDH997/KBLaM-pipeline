# KBLaM-pipeline

KBLaM - original KBLaM source code with some edits in experiments/eval.py.

run-kblam-emb.sh - script for preparing embeddings for dataset.

run-kblam-eval.sh - script for running experiments via eval.py.

evaluate_results.ipynb - code used for quality evaluation.
